# NoStorageRestrict

![](https://i.imgur.com/Z7VH0Li.jpg)

Removes the restriction when selecting folders (Downloads/Android) through the file manager on Android 11 and higher

# Requirements
- Magisk 20.x
- Android 11 or higher
